# Барахолка Плюс

Интернет‑магазин для продажи новых и б/у вещей для частных лиц и компаний.

## Описание

«Барахолка Плюс» — это платформа для покупки и продажи новых и бывших в употреблении товаров. Сайт предоставляет удобный интерфейс для поиска, размещения объявлений и совершения покупок.

## Функциональность

* Главная страница с акциями и категориями товаров
* Каталог товаров с фильтрами и сортировкой
* Карточки товаров с фото, описанием и отзывами
* Личный кабинет пользователя с историей заказов
* Система размещения объявлений для продавцов
* Корзина покупок и оформление заказа
* Поиск по сайту
* Мобильная адаптация

## Технологии

* **HTML5** — структура страниц
* **CSS3** — стилизация и адаптивный дизайн
* **JavaScript** — интерактивность и динамические элементы
* **Чистый код** без использования фреймворков для максимальной производительности
